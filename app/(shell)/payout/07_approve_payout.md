## Audit Log

The Approve step displays the audit history for the payout run.

The audit log records all payout adjustments made during the Review step.

Each record includes:

* which professional name + id
* fee category affected
* adjustment amount
* reason
* user who made the change
* timestamp

Example:

Sarah Lee TF12321
Chat fee adjusted +$20
Reason: Missing chat record
Adjusted by Michael John
2:34 PM

The audit log allows clinical ops to verify all payout changes before final approval.
